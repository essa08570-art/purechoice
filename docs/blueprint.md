# **App Name**: PureChoice

## Core Features:

- Conversion-Optimized Hero: High-impact visual entry point featuring primary call-to-actions and clean branding.
- Dynamic Product Grid: Responsive layout of product cards highlighting prices, ratings, and featured affiliate links.
- AI Pro/Con Synthesizer Tool: Generative AI tool that analyzes product specifications and user sentiment to generate a succinct pros/cons summary.
- Intelligent Filtering: Real-time client-side search and category filtering for rapid product discovery.
- SEO Schema Generator: Automated generation of structured JSON-LD data and metadata for product rich snippets in search results.
- Interactive FAQ Module: Clean, accordion-style component designed to address common buyer queries and improve time-on-page.
- Redirect Analytics Bridge: Intermediary redirect system to track outbound affiliate clicks while maintaining clean URLs.

## Style Guidelines:

- Primary Color: Trustworthy Electric Blue (#2659D9) representing clarity and authority.
- Background: Clean Pearl White (#F7F8FB) to ensure a high-end, minimal editorial feel.
- Accent: Vibrant Sky Cyan (#17A9CC) for highlights and call-to-action hover states to provide energetic contrast.
- Main and headline font: 'Inter' for a machined, objective look that communicates transparency and modern tech aesthetics.
- Feather-light, thin-stroke SVG icons for product attributes and navigational cues.
- Bento-style grid layouts with consistent 2rem spacing to maintain a structured, premium catalog feel.
- Subtle y-axis shifts and gentle opacity fades when product cards enter the viewport for a polished user experience.
