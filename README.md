# Dynamic Fashions

This is a curated fashion essentials platform built with Next.js, Tailwind CSS, and Shadcn UI.

## Getting Started

To run this project locally:

1. **Prerequisites**: Ensure you have [Node.js](https://nodejs.org/) installed.
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Run Development Server**:
   ```bash
   npm run dev
   ```
4. **Open the App**: Navigate to `http://localhost:9002` (or the port specified in your terminal).

## Project Features

- **Curated Collection**: High-quality fashion items selected by hand.
- **New Arrivals**: A dedicated filter for the latest style drops.
- **AI Sentiment Analysis**: Powered by Genkit to summarize user reviews (requires API keys).
- **Responsive Design**: Mobile-first approach using Tailwind CSS.
- **SEO Optimized**: Includes schema markup for products.

## Technologies Used

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Components**: Shadcn UI / Radix UI
- **Icons**: Lucide React
- **AI**: Genkit (Google Gemini)
